# README

## Project Title

**LeetCode Array Challenge**

## Objective

The objective of this project is to practice and improve problem-solving skills by implementing common array operations in C. The project focuses on searching, sorting, and analyzing array elements while understanding the efficiency of different algorithms.

## Problem Statement

Develop a C program to solve array-based challenges that include:

* Finding the largest element.
* Finding the second largest element.
* Finding the smallest element.
* Finding the second smallest element.
* Reversing the array.
* Removing duplicate elements.
* Counting the frequency of each element.
* Performing searching operations.
* Sorting the array.
* Analyzing the time complexity of each operation.

## Features

* Identifies the largest and second largest elements.
* Identifies the smallest and second smallest elements.
* Reverses the elements of an array.
* Removes duplicate values from the array.
* Counts the frequency of each element.
* Performs searching operations on the array.
* Sorts the array in ascending or descending order.
* Includes time complexity analysis for each operation.

## Requirements

* C Compiler (GCC, Turbo C, Code::Blocks, Dev-C++, etc.)
* Any operating system supporting C programming.

## Time Complexity Analysis

| Operation            | Time Complexity                         |
| -------------------- | --------------------------------------- |
| Find Largest         | O(n)                                    |
| Find Second Largest  | O(n)                                    |
| Find Smallest        | O(n)                                    |
| Find Second Smallest | O(n)                                    |
| Reverse Array        | O(n)                                    |
| Remove Duplicates    | O(n²) *(without extra data structures)* |
| Count Frequency      | O(n²) *(basic approach)*                |
| Linear Search        | O(n)                                    |
| Bubble Sort          | O(n²)                                   |

## Learning Outcomes

After completing this project, the user will be able to:

* Understand array manipulation techniques.
* Implement common searching and sorting algorithms.
* Analyze the efficiency of algorithms using time complexity.
* Improve logical thinking and problem-solving skills.
* Apply array operations in practical programming scenarios.

## Conclusion

This project provides hands-on experience with fundamental array operations and algorithm analysis. It strengthens programming skills by combining practical implementation with an understanding of time complexity, preparing learners for coding interviews and competitive programming challenges.
